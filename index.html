Minecraft: Bedrock Edition Skin Grabber

This tool lets you grab skins of Bedrock Edition players that have joined a Geyser-based server at least once.


FAQ

How does this work?

Just type in a gamertag in the format of the placeholder text and click "Grab skin". Within a few seconds, you will see the new skin.

You can then use "Save Image As" or similar methods to download the skin to your device.

Is this safe?

Yes. You don't need to install software, nor does this site use bugs to install malware.

Why does this need to be Geyser-specific?

Because of the way Minecraft: Bedrock skins are transmitted, we have to use Geyser's cache to retrieve the skin.
